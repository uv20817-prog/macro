# Macro — Screenshot + AI Description

AutoHotkey + Python: select screen area → screenshot → describe with Gemma 4.

## Usage
1. Run `run.vbs` (starts background listener)
2. Select screen area
3. Receives AI-generated description via Gemma 4 (Ollama)

## Requirements
- Python 3.10+
- Ollama with Gemma 4
- `pip install -r requirements.txt`
